# Beschrijving

Voor dit project heb ik een interactieve website ontwikkeld waarbij de gebruikerservaring centraal staat. Het doel was om een moderne website te bouwen die:
- toegankelijk blijft voor iedereen;
- goed werkt op mobiel, tablet en desktop;
- modern en interactief aanvoelt;
- ook zonder JavaScript bruikbaar blijft.

Tijdens dit project heb ik gewerkt met:
- Node.js
- Express
- LiquidJS
- HTML/CSS
- Client-side JavaScript
- API-koppelingen

Ik heb stap voor stap functionaliteiten toegevoegd en verbeterd via GitHub commits. Hierdoor is goed te zien hoe het project zich heeft ontwikkeld van een simpele basiswebsite naar een moderne interactieve gebruikerservaring.

---

## Progressive Enhancement

In dit project heb ik gebruikgemaakt van Progressive Enhancement.

Dat betekent:
1. Eerst een sterke HTML basis.
2. Daarna styling met CSS.
3. Vervolgens extra interactie met JavaScript.

De website werkt daardoor nog steeds zonder JavaScript.

JavaScript wordt gebruikt als enhancement bovenop de basisfunctionaliteit.

# View Transitions

Ik heb gebruikgemaakt van de View Transitions API om vloeiende pagina-overgangen toe te voegen.

Hierdoor:
- verlopen navigaties soepeler;
- voelt de website moderner aan;
- krijgt de gebruiker een betere ervaring.

De View Transitions zijn toegevoegd als Progressive Enhancement:
- browsers die het ondersteunen krijgen animaties;
- andere browsers behouden normale navigatie.

---

# Loading State

Naast de server-side functionaliteit heb ik client-side JavaScript gebruikt om loading states toe te voegen.

Wanneer een gebruiker een actie uitvoert:
- wordt de knop tijdelijk uitgeschakeld;
- verandert de tekst naar bijvoorbeeld:
  - “Loading...”
  - “Adconnect logo...”;
- krijgt de gebruiker directe feedback.

Hierdoor voelt de website sneller en professioneler aan.

# Success State

Na een succesvolle actie krijgt de gebruiker een success state te zien.

Ik heb hiervoor extra client-side JavaScript toegevoegd zodat gebruikers direct visuele feedback krijgen wanneer een actie succesvol is uitgevoerd.

Bijvoorbeeld:
- succesmeldingen;
- dynamische feedback;
- duidelijke bevestiging;
- animaties.

## Confetti Effect

Voor mijn success state heb ik een confetti animatie toegevoegd met client-side JavaScript.

Hierdoor:
- voelt de interactie moderner aan;
- krijgt de gebruiker duidelijke feedback;
- voelt een succesvolle actie leuker en interactiever.

De confetti wordt bijvoorbeeld getoond:
- na het succesvol versturen van een formulier;
- na een succesvolle interactie;
- als visuele bevestiging.

# Installatie

## 1. Clone de repository

```bash
git clone https://github.com/JasinAhmed/user-experience-enhanced-website.git
```

## 2. Open de projectmap

```bash
cd user-experience-enhanced-website
```

## 3. Installeer dependencies

```bash
npm install
```

## 4. Start de server

```bash
npm start
```

## 5. Open in browser

```bash
http://localhost:8000
```
